# 座標檢視器 Coordinate Viewer

LCH 微孔加工座標的視覺化與量測分析工具。可載入各區域的微孔座標、依加工順序排列、檢視分布，並把台超量測產出的結果與超出規格的位置標示在座標圖上。

提供兩個版本，核心功能一致：

- **網頁版** `coordinate_viewer.py`（Streamlit + Plotly）
- **桌面版** `coordinate_viewer_desktop.py`（PyQt5 + pyqtgraph，深色介面、可流暢處理上萬點）

> 兩版都有：**同時載入多個量測檔**（每檔一個分頁）、**.csv 與 .xlsx**、載入後**依矩形編號排序**、載入失敗的**詳細原因提示**、量測分析的 **各項分佈圖**、疊合圖的 **分面板錨點對位**、以及 **矩形 ↔ 孔 雙向對照鎖定**。
>
> 差異只在「設錨點」的操作方式：桌面版是**點選**（點量測矩形 → 點對應孔 → 按「設面板錨點」）；網頁版因 Streamlit 每次點擊會整頁重跑，改用**輸入框**（輸入矩形編號 + 選區域 + 孔序號）。

## 功能

- 載入多個座標 CSV，合併或分檔檢視；每個區域以高對比顏色區分。
- 依 `DrillDataSet` 的加工順序排列各區域（依檔名比對；名稱完全對不上會跳對話框說明，不會靜默）。
- 點圖上的點或表格列即可鎖定、放大置中並紅圈標示；滾輪以游標為中心縮放、可拖曳平移。合併顯示點圖時，右側檔案清單會自動展開並捲到對應列。
- 翻轉到背面（X 鏡像）、繞原點逆時針旋轉。
- 全部顯示/隱藏；依序隱藏動畫（載入 DrillDataSet 後才會出現此按鈕，依其加工順序播放）。
- 量測檔分析：解析台超量測原始檔（.csv／.xlsx），列出資料表與 X/Y 中心分布圖，載入後依矩形編號排序。自動判斷編碼與分隔符號，UTF-16／Tab 原始檔或被 Excel 另存的 Big5／逗號 CSV 都能載入。
- 各項分佈圖：勾選任意欄位，把各欄以不同顏色畫在同一張圖（桌面版可點圖/點列表鎖定、放大、雙向同步）。
- 超出規格標紅：X/Y 中心以「量測值−CAD值」的偏差比上下限；間距/高度以「量測值」直接比上下限。各項可分別設上限/下限，填 0 = 不檢查該方向。
- 疊合圖：把超規紅點疊到座標檢視器彩色區域上，採**分面板錨點對位**——量測資料通常是上下兩塊面板，因孔陣列密集且對稱、電腦無法自動分辨正確相位，需為每個面板各指定一個錨點（一個量測矩形 ↔ 對應孔），系統再依錨點把各面板精準對齊。設好每個面板的錨點後才會解鎖「產生疊合圖」（桌面版用點選、網頁版用輸入框設錨點）。
- 矩形 ↔ 孔 **雙向**對照：疊合完成後，點量測矩形 → 合併顯示鎖定對應孔；反之點合併顯示的孔 → 量測分頁鎖定對應矩形，方便對照「矩形編號 ↔ 哪一區的哪個孔」。
- 載入失敗提示：檔案讀不進去時說明原因（編碼／分隔符號、缺 X/Y 欄、非量測原始檔等），桌面版以對話框呈現。

## 原理

**座標系對齊（疊合圖）**

量測結果與座標檢視器畫的是同一片板子的同一批孔，但座落在兩個不同的座標系：量測是機台量出來的座標（含旋轉、翻面造成的鏡像、平移），設計座標是 CSV 裡的孔位。兩者的**孔間距相同**（同尺度），因此它們之間只差一個**剛體轉換**（旋轉＋水平翻轉＋平移），不需要縮放。

- **自動配準**：先用「格點佔用率」粗搜最佳旋轉角與是否水平翻轉，再用 **ICP（最近點迭代）＋ Kabsch 最小平方**把轉換精修到次微米等級（殘差約 0.0005 mm）。
- **為什麼需要錨點**：微孔陣列**又密又對稱**，而且量測通常只涵蓋部分孔。這會造成「相位歧義」——把整塊面板平移整數個孔距，點一樣會落在孔上（佔用率甚至更高），但對應到的是**錯的孔**。純靠「最多點落在孔上」會選到錯的相位，常見結果是「上半塊對、下半塊整排位移」。這種歧義**數學上無法只靠幾何自動分辨**。
- **分面板錨點對位**：解法是引入一個已知的對應點。使用者為每個面板各指定一個**錨點**（一個量測矩形 ↔ 它對應的孔），程式把該面板的剛體轉換**初始化在錨點上**再做 ICP 收斂——錨點鎖住正確相位，ICP 負責把整塊貼準。每塊各一個錨點即可。

**超規判定**

- X/Y 中心：偏差 =「量測值 − CAD值」，超過上限或低於下限即標紅。
- 間距/高度：直接用「量測值」比上限/下限。
- 任一欄填 0 = 不檢查該方向；任一項超出，該矩形即標紅。

**矩形 ↔ 孔 對照**

有了各面板的轉換後，把量測矩形的座標轉到座標檢視器座標系，取**最近的孔**，即得「矩形編號 → 哪一區、第幾個孔」。這個對照是**雙向**的：點量測矩形會鎖定合併顯示的孔；反過來點合併顯示的孔，也會用同一組轉換反查最近的量測矩形並鎖定量測分頁。

## 安裝與執行

需先安裝 [Python](https://www.python.org/downloads/)（安裝時勾選 *Add Python to PATH*）。

Windows 使用者可直接雙擊批次檔，第一次執行會自動安裝套件：

- 網頁版：`start.bat`
- 桌面版：`start_desktop.bat`

或手動安裝後執行：

```bash
pip install -r requirements.txt

# 網頁版
streamlit run coordinate_viewer.py

# 桌面版
python coordinate_viewer_desktop.py
```

## 輸入檔案格式

- **座標 CSV（可多選）**：各區域的微孔座標，含 `x`、`y` 欄位（例：`69x71_250um-aliceblue.csv`）。
- **DrillDataSet（選用）**：加工區域順序，含 `ArrayName` 或 `FilePath` 欄，可自行調換順序。其中的區域名需與載入的座標 CSV 檔名一致才對得上；讀不到欄位或名稱完全對不上時會跳對話框說明。載入後才會出現「依序隱藏」按鈕。
- **量測原始檔（選用，可多選）**：台超量測產出的原始檔，含「矩形 N: X 中心座標 …」等群組資料。支援 **.csv 與 .xlsx**；CSV 原始格式為 UTF-16／Tab 分隔，被 Excel 另存成 Big5／逗號或存成 .xlsx 也能自動辨識載入。

## 檔案說明

```

coordinate_viewer_desktop.py  桌面版主程式
start.bat                     網頁版啟動（自動安裝套件）

requirements.txt              相依套件
```

> 實際的量測資料、座標 CSV 與產出檔案已由 `.gitignore` 排除，不會上傳到版本庫。
